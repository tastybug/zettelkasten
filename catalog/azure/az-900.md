# AZ-900

## Account, Subscription, Resource Groups

At the root, there is the `Azure Account`, which contains one or more `Subscriptions`.
Subscriptions contain one or more `Resource Groups`, which then contain the actual cloud resources.

`Account 1-n Subscription 1-n Resource Group 0-n Resource`

### Subscription

Subscription are a unit of management, billing and scale. You can use the subsciptions to configure different billing models and apply different access-management policies.

It comes with 2 types of boundaries:

1. Billing Boundary: this type determines how an Azure account is billed. Azure generates separate billing repports and invoices so that you can organize and manage costs.
2. Access Control Boundary: Azure applies access-management policies at the subscription level and you can create separate subscriptions to reflect different organizational structures (akin to having different departments).
